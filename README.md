# ✨ Prompt Gallery

A curated collection of interactive web experiences crafted with elegance and sophistication.

**Live Site:** https://gloceahn-ux.github.io/prompt-gallery/

---

## 🎨 Projects

### 1. **Luxury Infinite Facts+**
Discover fascinating facts from across the universe with style and elegance.

**Features:**
- 🎲 Dynamic fact generation using AI-style templates
- 🌍 5 categories: Space, Animals, Earth, Human Body, All
- 🌙 Dark/Light theme toggle with persistent storage
- ❤️ Favorites system (saved to localStorage)
- ✨ Animated particle background
- 🔊 Sound effects on fact generation
- 📱 Fully responsive design

**Visit:** [Luxury Infinite Facts+](./luxury-facts/index.html)

---

### 2. **PRRD Infographic Facts**
Educational information about the 16th President of the Philippines presented in an elegant three-panel layout.

**Features:**
- 📋 Three-panel infographic design (Key Facts | Feature Fact | Strengths)
- 🎯 Interactive fact generation
- 💡 Key information and strengths highlights
- 📌 Inspirational quote section
- 🎨 Luxury aesthetic with smooth transitions
- 📱 Mobile responsive layout

**Visit:** [PRRD Infographic Facts](./prrd-facts/index.html)

## 🛠️ Technology Stack

- **HTML5** — Semantic structure
- **CSS3** — Modern styling with gradients, animations, and backdrop filters
- **JavaScript (Vanilla)** — Interactive functionality
- **Canvas API** — Animated particle background (Luxury Facts+)
- **Web Audio API** — Sound effects
- **localStorage** — Data persistence

## 🚀 Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/gloceahn-ux/prompt-gallery.git
   cd prompt-gallery
   ```

2. Open in your browser:
   - **Landing Page:** Open `index.html` locally or visit the live site
   - **Luxury Facts:** Open `luxury-facts/index.html`
   - **PRRD Facts:** Open `prrd-facts/index.html`

3. No dependencies or build process required — pure HTML/CSS/JS!

---

## 💾 Data Persistence

**Luxury Infinite Facts+** uses browser localStorage to save:
- Theme preference (dark/light)
- Favorite facts
- Browsing history (last 10 facts)

Data is stored locally and persists across browser sessions.

---

## 🎨 Customization

### Colors
All projects use CSS variables for easy theming. Update the gold color in styles:
```css
color: #d4af37;  /* Change this to customize */
```

### Facts
Modify the facts arrays in the `<script>` sections of each project.

### Templates
In **Luxury Infinite Facts+**, edit the `templates` object to change fact generation patterns.
